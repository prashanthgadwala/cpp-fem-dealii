Introduction

The second coding assignment is used to get familiar with tensor calculus related deal.II class templates. You should therefore use the online documentation:

Online documentation of deal.II

Today we will work with the class templates

Tensor<1,dim>
Tensor<2,dim>
Tensor<4,dim>
Vector<double>
FullMatrix<double>

from the finite element library deal.II, whereby "dim" is a template variable which allows to vary e.g. between the two- and three dimensional case (remember from the last exercise the basics of function and class templates). As described in the brief repetition of the essentials in C++, the respective tensor class templates allow different dimensions as input, i.e. dim=1,dim=2,dim=3 for each respective rank. To be able to use those files, they need to be included, i.e.

#include <deal.II/lac/full_matrix.h>
#include <deal.II/lac/vector.h>
#include <deal.II/base/tensor.h>
#include <iostream>

As mentioned in the brief documentation of C++ we are using frequently deal.II class templates and functions. These are grouped into the namespace dealii. Hence would have to write for instance

dealii::Tensor<1,dim> testensor;

to declare an instance of the rank-one tensor class template. Therefore we once tell the compiler that we intend to use this namespace via

using namespace dealii;

which allows to omit the prefix dealii::, i.e.

Tensor<1,dim> testensor;
