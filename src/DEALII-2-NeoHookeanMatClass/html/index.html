Introduction

In the second Coding Assignment the previously covered tensor class templates are used to compute different strain and stress measures for Neo-Hookean hyperelasticity. For now a deformation gradient \( \mathbf{F} \) is given and passed to an instance of the class "NeoHookeanMaterial". Your task is to complete the code in the marked areas using the definitions discussed in the class and during the exercise. Look out for the following

BEGIN - INSERT YOUR CODE HERE

END - INSERT YOUR CODE HERE

which indicates where you should complete the code. Within every member function a default instance of a tensor is already declared. Your task is to implement a proper definition, i.e. compute the respective quantites. Thereby a deformation gradient is provided from the main()-function.

As always make use of the deal.II online library deal.II

Computing different strain measures

Your first task will be to open the provided C++ file and search for the definition of the following functions that are grouped in the namespace StrainMeasures in the header file StrainMeasures.h. Furthermore, consider the functions that are in the namespace Physics::Elasticity from deal.ii, which provide for instance a dim-dimensional symmetric unit tensor, e.g. Physics::Elasticity::StandardTensors<dim>::I.

StrainMeasures::get_RightCauchyGreenTensor(const Tensor< 2, dim > &F);
StrainMeasures::get_LeftCauchyGreenTensor(const Tensor< 2, dim > &F);
StrainMeasures::get_GreenLagrangeTensor(const Tensor< 2, dim > &F);
StrainMeasures::get_AlmansiTensor(const Tensor< 2, dim > &F);

and complete the computations of the various strain measures using the deformation gradient as described.

Computing different stress measures
