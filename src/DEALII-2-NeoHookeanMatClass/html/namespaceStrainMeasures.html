Detailed Description

Functions to compute several strain measures grouped in a namespace.

  • Author
    Dominic Soldner
    Version
    3.0
    Date
    Jan 2021
    A namespace that groups several functions to compute different stress measures, using a 2nd order tensor, i.e. the deformation gradient, as input. Standard unit tensors are used from the header file deal.II/physics/elasticity/standard_tensors.h.
